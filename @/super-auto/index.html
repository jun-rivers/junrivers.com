スーパーオート

Super-Auto

使用説明

Instructions

  1. ページの上左の方にある漢字を書いて下さい
    Try to write the character as shown on the top-left corner.
  2. 出来るだけ、綺麗に書いて下さい
    Write as beautifully as possible.
  3. 字を完成したら、「次へ」を押し、次の字を書き始めてください
    Once completed, click 'next', and start writing the next character.
  4. 5分の中で出来るだけたくさんの字を書いて下さい
    Finish as many characters as possible within 5 minutes.
